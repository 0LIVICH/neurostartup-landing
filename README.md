# Landing Page Project

Лендинг для NeuroStartUp с современным дизайном и адаптивной версткой.

## Технологии
- HTML5
- CSS3
- Bootstrap 4
- JavaScript

## Структура проекта
- `css/` - стили и Bootstrap
- `js/` - JavaScript библиотеки
- `img/` - изображения и иконки
- `index.html` - главная страница
